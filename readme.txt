## Steps to setup the system.

1. Import as Maven Project
2. Create Database as 'mtit_assignment2' in MySQL
3. Configure MySQL properties in src\main\resources\application.properties
4. Run the project as Java Project
5. Visit http://localhost:8080/intialize to inialize database with sample data
6. http://localhost:8080/ will be ur home page
